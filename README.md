Hkkr.in is a [Curl](https://github.com/curl/curl) client for [Hacker News](https://news.ycombinator.com/)</br></br>
I spend a lot of time in the terminal, and also lots of time on HN, so I made hkkr.in to make viewing HN in the terminal simple and accesable from any computer, not just ones that have [HN CLI](https://github.com/search?q=Hacker+News+CLI) programs installed. I was inspired by the [wttr.in](https://github.com/chubin/wttr.in) which does a similar thing for the weather and for moon phases.
### Usage

To get the top ten posts, run `curl hkkr.in`
